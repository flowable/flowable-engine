Flowable Release Notes

Release Notes - Flowable - Version 5.22.0

Highlights

  • First Flowable release. Except from the different Maven group (org.flowable) and artefact id (flowable-engine etc) no changes have been made to package names or config file names.
  • Introduction of transient variables. Transient variables won't be persisted in the Flowable variable tables, but are still available on the execution for the duration of a single transaction. Transient variables can for example be used to store a REST response that is only needed to create a request to another REST service, or to use a Java service task result in the next Java service task without it getting persisted. You can read more about it here
  • Several bug fixes and smaller enhancements
